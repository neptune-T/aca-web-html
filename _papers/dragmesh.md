summary: We present DragMesh, a robust framework for real-time interactive 3D articulation. It decouples semantic intent reasoning (joint type) from geometric regression (axis/origin via KPP-Net), and uses a Dual Quaternion VAE with FiLM-conditioned non-autoregressive Transformer decoding to generate plausible, kinematically-consistent motion trajectories in real time.
